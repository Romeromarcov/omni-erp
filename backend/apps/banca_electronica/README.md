# App `banca_electronica`

Banca electrónica: gestión de las cuentas bancarias de la empresa para integración con banca en línea. Punto de partida para conciliación y pagos electrónicos.

**Prefijo API:** `/api/banca-electronica/`

## Modelos

| Modelo | Descripción |
|---|---|
| `CuentaBancariaEmpresa` | Cuenta bancaria de la empresa para banca electrónica. |

## Endpoints

Recurso REST (CRUD vía router): `cuentas-bancarias-empresa/`.
