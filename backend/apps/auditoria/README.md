# App `auditoria`

Log de auditoría transversal del sistema: registra acciones relevantes para trazabilidad y cumplimiento.

**Prefijo API:** `/api/auditoria/`

## Modelos

| Modelo | Descripción |
|---|---|
| `LogAuditoria` | Registro de auditoría (actor, acción, entidad, timestamp). |

## Endpoints

Recurso REST (CRUD vía router): `logs-auditoria/`.
